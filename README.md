## 🔍DATA LENS 🔍
### _Application For Air Control Quality in Indonesia's Big Cities_ <br/>
  DataLens is a web dashboard that displays real-time air quality data for major cities in Indonesia. 
  Its purpose is to help the public monitor daily pollution risks before engaging in outdoor activities. <br/>
